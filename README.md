# family-archive-appliance

가족 사진을 업로드, 조회, 보관하기 위한 개인용 self-hosted 패밀리 앨범 서버 프로젝트입니다.

완제품 NAS를 그대로 사용하는 대신, 저전력 x86 기반 홈서버 위에 Linux와 Docker Compose를 구성하고, 패밀리 앨범에 필요한 기능만 직접 구현하는 self-hosted appliance를 목표로 합니다.

---

## 배경

가족 사진은 시간이 지날수록 여러 장치와 서비스에 흩어지기 쉽습니다. 휴대폰, PC, 메신저, 클라우드, 외장 저장장치에 사진이 분산되면 어디에 있는지 찾기 어렵고, 가족이 같은 사진을 함께 보기도 어렵고, 백업 여부를 확인하기도 어렵습니다.

이 문제를 해결하기 위해 가족 사진을 한 곳에 모으고 웹에서 조회할 수 있는 작은 시스템을 직접 만들기로 했습니다.

완제품 NAS도 선택지였습니다. 하지만 Docker 기반 커스텀 서비스를 자유롭게 운영하고, 이후 홈서버 플랫폼으로 확장하는 것까지 고려하면 직접 구성하는 방향이 더 맞다고 판단했습니다.

---

## 이 프로젝트가 하는 것

주 목적은 패밀리 앨범 서버를 만드는 것입니다.

- 가족 사진 업로드
- 사진 목록 및 날짜별 조회
- 썸네일 생성
- 원본 파일 보존
- 삭제 보존 정책

부 목적은 개인용 홈서버 플랫폼을 확보하는 것입니다.

- Linux 홈서버 운영 경험
- Docker Compose 기반 self-hosted 서비스 구성
- 스토리지 정책과 백업 정책 직접 설계
- 사이드 프로젝트 운영 환경 확보

## 이 프로젝트가 하지 않는 것

초기 버전에서는 다음을 하지 않습니다.

- RAID 구성 또는 다중 HDD 자동 관리
- 얼굴 인식, 객체 인식
- 완전 자동 백업
- 모바일 앱 또는 WebView App
- 외부 공개 서비스
- 복잡한 사용자 권한 모델
- 상용 NAS 수준의 관리 UI

초기 버전은 다음 한 문장으로 정의합니다.

> 가족 사진을 안전하게 업로드하고 조회할 수 있는 최소한의 self-hosted album server를 만든다.
